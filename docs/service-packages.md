# Fiverr Service Packages

These packages mirror the Fiverr listing for automated WordPress design delivery. Each level builds on the prior and includes automation-ready steps, tools, and handoff criteria.

## Basic — Starter Launch Page ($85)
- **Scope:** 1-page responsive WordPress landing page with branded hero, services/features section, testimonials, and contact form.
- **Automation focus:**
  - Use a starter template (Elementor/GeneratePress + child theme) with pre-made blocks.
  - Pre-configure a WP-CLI script to install theme, contact form plugin, SEO basics, and caching.
  - Intake form auto-maps brand colors, logo, and CTA text into a JSON variables file consumed by the template importer.
- **Deliverables:** Functional page, mobile responsiveness checks, contact form routing to client email, 1 revision. Ready-to-export backup via `wp-cli export`.
- **Turnaround:** 3-day delivery.

## Standard — Business Website ($175)
- **Scope:** 3-page site (Home, About/Services, Contact) with forms, SEO basics, and integrations.
- **Automation focus:**
  - WP-CLI provisioning script adds SEO/caching/security plugins and creates starter pages.
  - Form builder preset with opt-in checkbox and CRM/webhook integration.
  - Autogenerated on-page SEO checklist (meta titles/descriptions, alt text) via a task checklist template.
- **Deliverables:** Branded 3-page responsive site, working forms, 2 revisions, basic analytics (GA4/Matomo) hook, and a labeled site map PDF.
- **Turnaround:** 5-day delivery.

## Premium — Brand Experience Site ($355)
- **Scope:** 5-page site with animation accents, shop/catalog section, SEO uplift, and performance tuning.
- **Automation focus:**
  - Deployment script installs WooCommerce or a lightweight catalog plugin with demo products to accelerate setup.
  - Prebuilt animation library (CSS/GSAP snippets) selectable via data attributes to keep build time low.
  - Speed optimization recipe: CDN toggle, image compression workflow, and server/asset caching profile.
- **Deliverables:** 5-page responsive site with shop integration, payment gateway configuration, speed optimizations, 3 revisions, analytics, and an exportable staging-to-production migration plan.
- **Turnaround:** 7-day delivery.

## Add-ons
- Ongoing care plan (updates/backups).\
- Extra revision blocks.\
- Copy polish and image sourcing bundles.\
- Additional integrations (e.g., Zapier, email marketing, booking tools).

## Delivery Checklist (applies to all tiers)
- ✅ DNS or staging URL confirmed.
- ✅ WP core + required plugins installed via WP-CLI script.
- ✅ Theme child + brand tokens (colors, fonts, logos) injected.
- ✅ Forms tested with success and failure responses.
- ✅ PageSpeed/GTmetrix snapshot captured.
- ✅ Admin handoff video + credentials rotated.

## Files to keep in this repo

- `/templates/` for starter blocks and brand token JSON.\
- `/checklists/` for delivery QA across packages.
